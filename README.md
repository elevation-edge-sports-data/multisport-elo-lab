# MultiSport Elo Lab — Version 6

## Overview

MultiSport Elo Lab is a modular, configuration-driven Elo rating and simulation platform designed for:

- Sports modeling
- Historical backtesting
- Monte Carlo simulation
- Parameter optimization
- Model evaluation

Version 5 established the core Elo engine architecture through a unified execution pipeline, configuration-driven model behavior, and reusable modeling workflows.

Version 6 introduces an interactive Streamlit dashboard layer built on top of the existing Elo Lab engine, providing visual interfaces for simulation, Elo ratings, Elo evolution, evaluation, and exploration of model outputs.

---

# Version 6 Dashboard

Version 6 adds a Streamlit application layer for interacting with outputs generated by the Elo Lab modeling workflows.

The dashboard provides five analytical views:

---

## Model Configuration

The Model Configuration tab provides a summary of the current dashboard configuration.

Displays:

- Selected sport and season
- Simulation count
- Active model components
- Current model configuration summary

Interactive parameter controls are planned for a future version.

---

## Season Simulation

The Season Simulation tab executes Monte Carlo season simulations and provides interactive exploration of simulation outputs.

Features include:

- Configurable Monte Carlo simulation count
- Expected win summaries
- Team-level win distributions
- Expected wins
- Median wins
- Standard deviation
- 90% simulation ranges
- Interactive probability distributions

---

## Elo Ratings

The Elo Ratings tab visualizes the latest postgame Elo ratings from historical backtest results.

Features include:

- Current team Elo rankings
- Conference filtering
- Division filtering
- Top-team Elo visualization

---

## Elo Evolution

The Elo Evolution tab provides historical and simulated Elo trajectory analysis.

Features include:

- Historical weekly Elo evolution
- Multi-team Elo comparisons
- Simulated Elo trajectories from Monte Carlo simulations
- Mean simulated Elo paths
- 5th–95th percentile uncertainty bands

---

## Model Evaluation

The Model Evaluation tab provides comparative analysis of Elo model performance.

Metrics include:

- Accuracy
- Log loss
- Brier score

The dashboard presents metric-specific leaders without automatically designating a single overall best model.
